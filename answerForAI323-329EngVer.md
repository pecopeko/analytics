## This Week's Mode
This is not a polishing week.
This is a `try very different Shorts/TikTok-native formats` week.

Recent platform direction we are following:
- TikTok is rewarding more real, rough, conversational, less-overproduced content
- TikTok itself recommends telling the same message in different formats instead of repeating one format
- Comments and co-creation matter more
- YouTube Shorts is leaning into riffable formats, debates, interactive/game-like videos, and relatable twists on familiar formats

So this week:
- fewer generic app ads
- more `debate`, `hot take`, `POV confession`, `interactive mini-game`, `duo challenge`, `IGL identity`
