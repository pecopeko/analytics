# 2026/03/15 時点の実行パック

## 役割
ここには `Codexだけで作るもの` だけを書く。

前提:
- 使える手段は `Soraで15秒動画を作る` と `投稿する` だけ
- 動画編集、画面録画、DM、ASO、細かい分析設定は前提にしない
- スクリーンショットの見た目を踏まえて、Soraで再現しやすい企画にする

## 今回の方針
- GoGoBombs は `配置 / 二択 / 頭脳戦 / スマホ完結`
- Gymty は `日付で探せる / 地域で絞れる / すぐ探せる`
- どちらも `便利です` ではなく `使う場面が一発で伝わる` 動画にする

## 優先順位
1. GoGoBombs: 4本
2. Gymty: 2本

## 1. GoGoBombs: 配置ゲーム訴求

### Soraプロンプト
```text
9:16の縦動画、15秒、日本語音声あり。スマホアプリ紹介のショート動画。広告っぽすぎず、ゲーム好きの個人投稿っぽい熱量。音あり前提。大きく読みやすい日本語テロップを少なめに入れる。既存ゲームのロゴ、固有UI、キャラクターは出さない。

画面の雰囲気は、暗めのネイビー背景のスマホゲームUI。上部に「ラウンド1」「アタッカー配置」のような文字。中央には白い通路と茶色い壁でできたシンプルなマップ。左上に赤いエリアB、右上に青いエリアA。下部には「エントリー」「リーコン」「スモーク」「センチネル」のような役割ボタンが並ぶ。全体はミニマルで見やすい。

構成:
0-2秒: 大きい文字「撃ち合いなしで、ここだけ考えたい」日本語音声も同じ。
2-6秒: スマホ上でマップを見ながら、アタッカーをどこに配置するか迷う。役割ボタンが光る。テロップ「配置で勝負」
6-11秒: 配置を決めると、画面がスムーズに試合進行モードへ切り替わる。マップ上のコマが進み、AとBどちらに寄るかで緊張感が出る。
11-15秒: 黒背景にスマホ。日本語音声で「頭脳戦だけやりたい人向け。」テロップは「頭脳戦だけやりたい人へ」「App Storeで『ごごばく』検索」

テンポは速め。最後の4秒はアプリ名を大きく見せる。
```

### 投稿文
```text
撃ち合いじゃなく、配置と判断だけやりたい人へ。App Storeで「ごごばく」検索。#戦略ゲーム #VALORANT #CS2 #スマホゲーム
```

## 2. GoGoBombs: 二択クイズ訴求

### Soraプロンプト
```text
9:16の縦動画、15秒、日本語音声あり。コメントしたくなるクイズ系ショート。音あり前提。大きい日本語テロップ。既存ゲームのロゴや固有UIは出さない。

スマホ画面の雰囲気は、暗い背景にシンプルなタクティカルマップ。白い通路、茶色い壁、左に赤いBエリア、右に青いAエリア。マップ上に複数の赤い味方コマがいる。

構成:
0-2秒: 大きい文字「この場面、A行く？B行く？」日本語音声も同じ。
2-7秒: マップ上で赤いコマが中央付近にいて、AかBか迷う状態。テロップ「人数有利」「時間少ない」
7-11秒: 画面が左右分割され、左に「A寄り」、右に「B寄り」。矢印でルートが光る。音声は「あなたならどっち？」
11-15秒: スマホに“判断だけ遊ぶ戦略アプリ”が映る。音声で「こういう二択だけ遊べる。」テロップは「こういう二択だけ遊べる」「App Storeで『ごごばく』検索」

全体はネイティブショート動画風で、賢そうに見せすぎず、すぐ意味が分かることを優先する。
```

### 投稿文
```text
A寄り？B寄り？ App Storeで「ごごばく」検索。#VALORANT #CS2 #戦術 #ゲームアプリ
```

## 5. Gymty: 日付で探せる訴求

### Soraプロンプト
```text
9:16の縦動画、15秒、日本語音声あり。実用系ショート。広告っぽすぎず、スマホで見てすぐ理解できる。大きく読みやすい日本語テロップ。音あり前提。

スマホ画面の雰囲気は、明るい白基調のアプリUI。上部に日付が丸く並び、下に施設カードが縦に並ぶ。カードには「バドミントン」「兵庫県」「姫路市」「18:00-21:00」のような情報が見える。やさしいブルーとオレンジを使った見やすいデザイン。

構成:
0-2秒: 大きい文字「明日どこ空いてる？」音声も同じ。
2-6秒: 複数の日付を指でスライドする。施設カードが切り替わる。テロップ「日付で探せる」
6-11秒: バドミントンの施設候補が複数出る。地域と時間帯がパッと見で分かる。音声は「この見え方なら早い」
11-15秒: 音声で「空き状況をすぐ見たい人向け。」テロップは「すぐ探せる」「App Storeで『Gymty』検索」

“実用だから使う”が一発で伝わるテンポにする。派手さより理解の速さを優先。
```

### 投稿文
```text
明日どこ空いてる？を早く知りたい人向け。App Storeで「Gymty」検索。#体育館 #バドミントン #便利アプリ #スポーツ
```

## 6. Gymty: 地域絞り込み訴求

### Soraプロンプト
```text
9:16の縦動画、15秒、日本語音声あり。生活の困りごとを解決する実用ショート。白基調のきれいなアプリUI。大きい日本語テロップ。音あり前提。

スマホ画面の雰囲気は、検索条件ページ。上部に「検索条件」。地方、都道府県、市区町村を選べる。市町村名が小さな角丸ボタンでたくさん並び、そのうち「姫路市」が選択されている。清潔感があり分かりやすい見た目。

構成:
0-2秒: 大きい文字「地域で絞れないと探せない」音声も同じ。
2-6秒: たくさんの市町村ボタンの中から、目的の地域を選ぶ。テロップ「地域で絞る」
6-11秒: その後、施設一覧に切り替わり、候補がすっきり表示される。音声は「欲しいのはこの絞り込み」
11-15秒: 音声で「近場だけ見たい人向け。」テロップは「近場だけ見たい」「App Storeで『Gymty』検索」

自治体サイトっぽい硬さではなく、スマホアプリとして軽く使える雰囲気にする。
```

### 投稿文
```text
近場だけ絞って探したい人へ。App Storeで「Gymty」検索。#体育館 #便利アプリ #バスケ #バドミントン
```

## 今回の判断
- GoGoBombs はスクリーンショットの `配置画面` と `進行画面` が一番差別化になっている
- Gymty はスクリーンショットの `日付横スクロール` と `地域絞り込み` が一番伝わりやすい
- だから、抽象的なブランド動画より `画面の特徴をSoraで再構成する動画` を優先する
