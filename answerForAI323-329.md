# 2026/03/23〜2026/03/29 実行パック

## 役割
ここには `Codexだけで作るもの` だけを書く。

前提:
- 使える手段は `Soraで10秒動画を作る` と `投稿する` だけ
- 動画編集、画面録画、DM、ASO、広告設定は前提にしない
- 全動画 `9:16 / 10秒 / 音あり / 冒頭1.5秒で結論`

## 今回の前提を変える
前回までの案は少し `普通` だった。

今週は `最近の TikTok / YouTube Shorts の空気` に寄せる。

最近の流れ:
- TikTok は `polishedすぎる広告` より `rough / real / spill the tea / 共感できる本音` が強い
- TikTok は `同じメッセージの繰り返し` より `同じテーマを別フォーマットで見せる` 方が良い
- TikTok は `コメントや共作っぽさ` が強い
- YouTube Shorts は `riffしやすい型`, `議論が起きる問い`, `ゲームっぽい参加感`, `日常POVのひねり` が強い

だから今週は
- ただのアプリ紹介を減らす
- `議論`, `挑発`, `POV`, `ミニゲーム`, `告白`, `幹事あるある` を混ぜる
- 1本ごとに `違うShorts文法` を試す
