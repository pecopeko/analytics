# 3/15〜3/21 投稿パック

## 前提
使えるリソースは `投稿` と `Soraで15秒動画を作る` だけ。

だから今週は、`Soraでそのまま完成品を出して、TikTokとYouTube Shortsに同時投稿する` だけに絞る。

配分は `GoGoBombs 5本 / Gymty 2本`。

理由:
- GoGoBombs は 2026/03/07〜2026/03/13 に `初回ダウンロード4`
- Gymty は同期間 `ダウンロード0`
- ただし Gymty も、抽象PRではなく `具体シーン` にすると再検証する価値はある

## 投稿ルール
- 毎日 `1本だけ`
- 同じ動画を `TikTok` と `YouTube Shorts` に同日投稿
- 動画は `9:16 / 15秒 / 音あり / 冒頭2秒で結論`
- Soraには下の文をそのまま入れる
- 投稿文も下の文をそのまま使う
