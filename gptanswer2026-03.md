# 2026年3月の結果を踏まえた考察

## 結論
3月の結果を見ると、両アプリで勝つ動画の文法がかなり違う。

- GoGoBombs は `世界観紹介` より `問い` が強い。
- Gymty は `便利です` より `今この条件で探したい` が強い。
- どちらも `きれいな広告` に寄せるほど弱くなりやすい。

4月のプロモーション動画は、`説明動画` ではなく `ユーザーが自分事にしやすい1問` に寄せた方がいい。

## GoGoBombs で良かった動画
- `撃ち合いじゃなくない?` のような意見先行
- `4v3。A行く?B行く?` のような二択
- `撃つよりコール考える方が好き` のような IGL アイデンティティ
- `エイム弱い。でも作戦考えるのは好き` のような confession

共通点:
- 冒頭1秒で `自分の立場` か `答えたくなる問い` がある
- 盤面や配置がすぐ出る
- アプリ説明は最後の2秒だけ

つまり GoGoBombs で良いプロモーション動画は、`このゲームを紹介します` ではなく、`あなたはこのタイプのプレイヤーですか?` と聞く動画。

## GoGoBombs で弱かった動画
- `3秒で選んで` のようなミニゲーム演出だけの動画
- `ボタンが気持ちいい` だけの UI フェチ動画
- `PC不要` のような広すぎるベネフィット
- 世界観だけ見せて、何を判断するゲームか曖昧な動画

理由:
- 参加ギミックだけでは `自分に関係ある` まで届きにくい
- 広い訴求は誰にも深く刺さらない
- このアプリは `判断のゲーム` なので、判断の場面を先に見せる必要がある

## Gymty で良かった動画
- `新宿区 バドミントン 空き 今日`
- `今日19時、空いてる?`
- `明日の夜だけ見たい`

共通点:
- 検索語そのものから始まる
- `今日`, `明日`, `19時` のように時間が具体的
- `体育館を探したい瞬間` がそのまま冒頭になっている

つまり Gymty で良いプロモーション動画は、`アプリ紹介` ではなく、`検索代行` に見える動画。

## Gymty で弱かった動画
- `体育館サイト、見づらくない?`
- 抽象的な `便利`
- 地域名も日時もない動画

理由:
- 不満訴求は、自分が今まさに困っている人にしか刺さらない
- Gymty は暇つぶしアプリではないので、広い共感より `今使う理由` が必要

## 4月に向いている動画の型

### GoGoBombs
1. 二択
- 例: `4v3。A行く?B行く?`

2. 意見
- 例: `タクティカルFPSで一番おもしろいの、撃ち合いじゃなくない?`

3. アイデンティティ
- 例: `撃つよりコール考える方が好き`

4. confession
- 例: `エイム弱い。でも作戦考えるのは好き`

### Gymty
1. 検索クエリ
- 例: `新宿区 バドミントン 空き 今日`

2. 今日ニーズ
- 例: `今日19時、空いてる?`

3. 明日ニーズ
- 例: `明日の夜だけ見たい`

4. 種目固定
- 例: `バドだけ見たい`

## 4月に止めるべき型
- 両アプリとも `広告っぽく整いすぎた紹介動画`
- GoGoBombs の `抽象ブランド訴求`
- Gymty の `既存Webへの文句`
- 1本の中に要素を詰め込みすぎる構成

## 4月の制作ルール
- 1本10秒
- 冒頭1.5秒は問いか検索語だけ
- 1動画1メッセージ
- CTAは毎回固定
- GoGoBombs は `問い→盤面→CTA`
- Gymty は `検索語→絞り込み→一覧→CTA`

## 評価の見方
3月の反省として、再生や `good` だけで勝ち負けを決めるとズレる。

4月は以下を主指標にした方がいい。

- 投稿後48時間の App Store impression
- first-time download
- その次に SNS の再生と反応

SNS の反応が強くてもインストールに落ちなければ、プロモーションとしては勝ちではない。
