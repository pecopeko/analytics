# 2026/03/23〜2026/03/29 For Human

## 役割
ここには `人間にしかできないこと` だけを書く。

今回の主題は `gogobombs を英語圏にちゃんと届けるための最低限の運用変更`。

## 先に結論
- 英語動画を日本語アカウントに混ぜるのは止める
- `TikTok` と `Instagram` は英語専用アカウントを分ける
- 英語アカウントでは `表示名 / bio / 投稿文 / 固定文` を全部英語にする
- 英語アカウントでは日本語投稿を混ぜない
- `YouTube` は multilingual 機能が使えるなら1チャンネル運用でもいい。使えないなら英語専用チャンネルを分ける

## なぜこれをやるか
- TikTok はアカウント作成時や変更時の `country or region` を、SIM や IP 情報などの位置情報で決めると公式に案内している
- Instagram は `Trial Reels` を使うと、まず non-followers に見せて試せると公式に案内している
- YouTube は multi-language audio と localized metadata を公式に用意している一方、視聴者の preferred language は `recommendations` そのものを切り替える設定ではないと公式に書いている。だから `英語タイトルにしただけ` では配信先の問題は解決しない

## 今週やること

### 1. 英語専用TikTokアカウントを作る
- 名前は `gogobombs` を含める
- bio は英語だけにする
- 投稿文、画面内文字、音声も英語だけにする
- 日本語投稿は一切混ぜない

### 2. 英語専用Instagramアカウントを作る
- 名前と bio を英語だけにする
- 英語版 7 本はこのアカウントで出す
- 投稿時に `Trial Reel` が使えるなら ON にする
- まず non-followers に当てて、反応が良いものだけ通常公開に回す

### 3. TikTok の `Account region` を確認する
- 変えられるなら、英語圏向けにしたいアカウント側で確認する
- ただし TikTok 公式では、地域変更にはその国・地域でのアクティビティなど条件がある
- 変えられないなら、無理にいじらず `英語専用アカウントを新規で分ける` 方を優先する

### 4. YouTube は分岐する
- `multi-language audio` や `automatic dubbing` が使えるなら、日本語動画に英語音声を付ける形もあり
- 使えないなら、英語専用チャンネルに英語動画を出す
- 中途半端に1つのチャンネルへ日英を交互に混ぜない

### 5. 英語投稿の型を固定する
- 英語版は毎回 `1 question + 1 outcome + 1 CTA`
- 例:
  - `4v3. Hit A or B?`
  - `Push or fall back?`
  - `Which setup loses?`
- 文化説明や長い前置きは削る

### 6. 英語版の固定CTAを揃える
- すべて `Search gogobombs on the App Store`
- `strategy-only mobile game`
- `for tactical FPS players`
- 毎回言い方を大きく変えない

## 今週やらないこと
- 日本語アカウントに英語動画を混ぜる
- 英語圏向けかどうか分からない状態で `英語弱い` と結論づける
- TikTok の地域変更だけを勝ち筋だと思い込む
